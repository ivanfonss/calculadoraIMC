# Calculadora de Índice de Massa Corporal (IMC)

Este é um programa Python simples que calcula o Índice de Massa Corporal (IMC) com base no peso e altura fornecidos pelo usuário. O IMC é uma métrica comumente usada para avaliar a saúde relacionada ao peso de uma pessoa.

## Como Usar

1. Certifique-se de que você tem o Python instalado em seu ambiente de desenvolvimento.

2. Abra um terminal e navegue até o diretório onde o arquivo `imc.py` está localizado.

3. Execute o programa Python com o seguinte comando:

   ```bash
   python imc.py


#O programa solicitará que você insira seu peso em quilogramas (no formato 75.300) e sua altura em metros (no formato 1.75).

#O IMC será calculado e exibido com duas casas decimais.

#O programa classificará o IMC em uma das seguintes categorias: abaixo do peso, peso normal, sobrepeso ou obeso.

##Exemplo de Saída

Digite o peso em quilogramas no formato 75.300 : 75.3
Digite a altura em metros no formato 1.75: 1.75
Seu IMC é: 24.65
Seu peso está normal.
